# Guia-de-Estudos-Para-Investimentos-Financeiros-Seguros-BRA-
Guia de estudos para aprender a investir de forma segura no Brasil, usando o NotebookLM.

Usando a ferramenta do NotebookLM da Google, foi feito um caderno temático tendo como assunto estudar investimentos seguros no mercado financeiro brasileiro. Como objetivos temos:
  1. Pesquisar métodos para investir no mercado brasileiro, de maneira segura;
  2. Aprender a construir um patrimônio durável, de maneira segura, através de investimentos no mercado financeiro brasileiro;
  3. Comparar os tipos de investimentos seguros que poderíamos ter no mercado financeiro brasileiro;
  4. Traçar planos de investimentos.

Como fontes, foram utilizados:
  1. Rendimento dos títulos no site do Tesouro Nacional: https://www.tesourodireto.com.br/produtos/dados-sobre-titulos/rendimento-dos-titulos
  2. Textos em pdf, com a descrição dos títulos possíveis de investimento no tesouro nacional, encontrados no site:
    a) https://www.tesourodireto.com.br/documents/d/guest/tesouro_selic
    b) https://www.tesourodireto.com.br/documents/d/guest/tesouro_ipca_juros_semestrais
    c) https://www.tesourodireto.com.br/documents/d/guest/tesouro_educa_mais
    d) https://www.tesourodireto.com.br/documents/d/guest/tesouro_renda_mais
    e) https://www.tesourodireto.com.br/documents/d/guest/tesouro_prefixado
  3. Live do palestrante Eduardo Moreira intitulada "Tudo o que você precisa saber antes de investir": https://www.youtube.com/live/BRauFmHa40I?si=DBg-9ufQgllU2V4A

Como prompts foram utilizados:
  1. "Mostre métodos para construir um patrimônio que seja durável, de forma segura, investindo no mercado financeiro no Brasil."
  2. "Considerando um médio prazo de 5 a 10 anos, a partir de 22 de junho de 2026, qual o título do tesouro nacional é mais vantajoso para investir?"
